# VueStore  

 项目演示地址：<http://129.204.233.33/VueStore>
 
## 技术栈

+ webpack
+ vue-router
+ vuex  

## 功能

+ 新闻资讯
  - 新闻资讯列表  
  - 新闻资讯列表详情  
  - 新闻评论功能  
+ 图片分享
  - 图片列表  
  - 图片缩略图功能
+ 商品购买
  - 经典商品列表布局  
  - 商品详情页面  
  - 添加购物车
+ 购物车功能

## 安装说明
1.克隆本仓库  

`git clone git@github.com:kejianlin/VueStore.git`  

2.安装依赖

`npm install`  

3.查看界面

`npm run dev`
